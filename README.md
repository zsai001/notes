## 📝 使用指南

1. **快速导航** - 使用 [知识索引](./知识索引.md) 快速找到相关内容
2. **模板使用** - 在 `templates/` 目录下找到各种笔记模板
3. **归档管理** - 将过时的内容移至 `archive/` 目录
4. **标签系统** - 使用标签方便分类和检索
5. **赚钱理财** - 使用专门的赚钱理财模板记录投资和创业思路

## 🎯 使用原则

- **持续记录** - 养成随时记录的习惯
- **定期整理** - 定期整理和更新内容
- **深度思考** - 不只是记录，更要思考和总结
- **实践应用** - 将学到的知识应用到实际中
- **财富积累** - 通过投资理财和副业项目实现财务自由
