# 快速开始指南

## 🚀 立即开始

### 1. 选择模板
根据您的需求选择合适的模板：
- 📖 **读书笔记** - 使用 `templates/读书笔记模板.md`
- 📚 **学习笔记** - 使用 `templates/学习笔记模板.md`
- 💭 **思考笔记** - 使用 `templates/思考笔记模板.md`
- 📋 **项目总结** - 使用 `templates/项目总结模板.md`
- 📅 **每日思考** - 使用 `templates/每日思考模板.md`

### 2. 创建笔记
1. 复制模板到相应目录
2. 重命名为有意义的文件名
3. 填写内容

### 3. 使用标签
在笔记末尾添加标签，方便后续检索：
```markdown
## 标签
#读书笔记 #技术 #Python
```

## 📁 目录使用建议

- 按技术栈分类
- 记录学习路径和进度
- 整理代码示例和最佳实践

### 哲学思考
- 记录日常思考
- 整理哲学观点

### 读书笔记
- 按阅读状态分类（已读/在读/待读）
- 记录阅读心得
- 建立知识连接

### 灵感收集
- 随时记录想法
- 定期整理和深化
- 转化为具体行动

2. **文件名规范** - 使用描述性的文件名
3. **目录导航** - 利用知识索引快速定位
4. **定期整理** - 定期整理和归档内容
