# 🚀 一键博客生成方案

## 📋 方案概述

我为您推荐并设置了一个基于 **Hugo** 的一键博客生成方案，具有以下特点：

- ⚡ **极快速度** - 几秒内生成整个网站
- 🎨 **精美主题** - 使用PaperMod主题，支持明暗模式
- 📝 **Markdown支持** - 完美支持您的笔记格式
- 🔄 **一键生成** - 自动化脚本，简单易用
- 🌐 **免费部署** - 支持GitHub Pages等免费平台

## 🛠️ 已创建的文件

### 核心脚本
- `generate-blog.sh` - 一键生成博客脚本
- `deploy-blog.sh` - 一键部署博客脚本

### 配置和模板
- `博客框架指南.md` - 详细的框架选择和使用指南
- `博客文章模板.md` - 各种类型的文章模板

## 🚀 使用方法

### 1. 首次使用
```bash
cd blog
./generate-blog.sh
```

### 2. 本地预览
```bash
hugo server
# 访问 http://localhost:1313
```

### 3. 部署到GitHub Pages
```bash
./deploy-blog.sh
```

## 📝 文章管理

### 创建新文章
```bash
hugo new posts/文章标题.md
```

### 文章结构
```
content/
├── posts/
│   ├── 技术学习/
│   ├── 哲学思考/
│   ├── 读书笔记/
│   └── ...
└── _index.md
```
